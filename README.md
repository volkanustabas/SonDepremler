# SonDepremler

- RestSharp api ile AFAD üzerinden son 5 gün içindeki 3.0 þiddetin üzerindeki depremleri listeler
- HtmlAgilityPack ile AFAD ve Kandilli Rasathanesi web sayfalarýndaki son depremleri listeler


## Kullanýlan Paketler

- SunnyUI
- RestSharp
- Newtonsoft.Json
- HtmlAgilityPack
